~ Project Title
HosteLink - Hostel Management System

~ Description
This is a web-based Hostel Management System designed to help manage hostel-related issues such as lost and found items, issue reporting, and general hostel coordination.
The project aims to improve communication and efficiency within the hostel environment.

~ Features
- User authentication
- Issue reporting system
- Lost and found management
- Annoucements system
- Informing caretakers about issues through emails
- Analytics
- Responsive UI

~ Tech Stack
- HTML
- CSS
- JavaScript
- Supabase
- n8n (workflow automation)
